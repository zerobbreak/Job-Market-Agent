================================================================================
JOB MARKET AI ANALYZER - CV PLACEHOLDER
================================================================================

This file serves as a placeholder for your CV file.

To use this application:

1. Replace this file with your actual CV in PDF format
2. Rename your CV file to "CV.pdf"
3. Make sure your CV contains your:
   - Contact information
   - Education background
   - Work experience
   - Skills and technologies
   - Career goals and objectives

The application will automatically detect and use CV.pdf if it exists in the
root directory.

For testing purposes, you can also specify a different CV file using:
python main.py --cv-path path/to/your/cv.pdf
